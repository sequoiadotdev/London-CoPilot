# London Copilot Frontend

This is the Next.js frontend for London Copilot.

For full project setup, API keys, backend instructions, and demo prompts, see the root repository README:

```text
../README.md
```

Local development:

```bash
npm install
npm run dev -- --hostname 0.0.0.0 --port 3001
```

Typecheck:

```bash
npm run typecheck
```

Keep real credentials in `.env.local` only. Do not commit API keys.
